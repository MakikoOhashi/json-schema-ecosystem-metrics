# AGENTS.md

This repository is a small GSoC proof-of-concept for JSON Schema ecosystem metrics.

Global rules:
- Keep scope minimal.
- Prefer one metric over multiple metrics.
- Do not rewrite unrelated files.
- Do not change API choices without explaining why.
- When editing, state exactly which files you changed.
- If blocked, propose the smallest viable workaround.
- For Part 2 evaluation, prioritize direct observations from actually running the code.
